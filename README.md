# ENGG200-ColoredHats

## Requirements
imutils==0.5.4

numpy==1.21.5

opencv-python==4.5.5.62
## Usage
python coloredhats.py [-h] [-v VIDEO] [-b BUFFER]

### required arguments:

  -v VIDEO, --video VIDEO
                        path to the video file

### optional arguments:

  -h, --help            show this help message and exit



  -b BUFFER, --buffer BUFFER
                        max buffer size, controls the movements trail's length
